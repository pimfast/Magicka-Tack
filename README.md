# Magicka-Tack

A game made for the Saxion CMGT intake 2026/2027.

SDG Theme: Quality Education (Goal 4)

I chose Quality Education because it's a highly important subject. Magicka-Tack is my attempt to make something that is engaging to play while quietly reinforcing the importance of education and having a choice in it.

*How to play*

    •    Download the zip file below
    •    Extract and run the .exe
    •    Windows only (GameMaker export limitation)
    
    •    Controls:
    •    Press B to launch ball
    •    Arrow keys to move
    •    Spacebar (or Z) to attack/confirm

*About the game*  
Magicka-Tack is a Breakout-inspired game with action reminiscent of Zelda.
You learn new things after each level, try to get a good enough score in all 5 of them to progress to see the end of the story!

*Built with*  
GameMaker Studio 2 / GML  
Grass Land tileset by Aleksandr Makarov
